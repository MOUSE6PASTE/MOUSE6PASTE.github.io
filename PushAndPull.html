About My Games

Push and Pull

The Push and Pull is a co-op only puzzle platformer about playing as two dualistic bunnies looking for a home

You can play on steam

It's a game about romance, relationships, closeness, and these peoples connections

I created this game from having this absolutely genius idea of a Push and pull co op game (This idea felt so intuitive to me I had to check it wasn't done already) and wanting to try using some code I made for multiplayer inputs from a project that fell through

The actual spark to start working on this idea was wanting go to a practice-your-pitch event and wanting to make something with my lovely girlfriend. After I did that I spent a month working on Wherever you get your podcasts then spent the next month making a demo and preparing for a showcase

The monitor we had set up had this little reflective base that, behind my laptop and various wires, I could just barely see the faces of the people behind the monitor. It made me happy to see one of those faces because they always had a smile. People have so much fun just pushing and pulling each other once they get the power. It's like they instantly start giggling when they start using it, I hadn't expected it to be so fun for people
